# Leanite
去我们的 [Leanite 开发日志](https://copws.github.io/leanite) 预览博客效果。

基于 LeanCloud 和 React 的动态博客系统，目前尚在测试。练手项目，有缘维护。

### 使用包

核心功能：

- React.js & React DOM
- React Router
- **UI：**Ant Design
- **LeanCloud：**leancloud-storage
- pubsub-js
- js-md5

Markdown 有关：

- **Markdown 编辑器：**for-editor
- Marked & highlight.js

同时感谢 for-editor 的 Markdown 文本渲染源码！（真的不会写……）
